Virtual Reading Room

Virtual Reading Room built with HTML, CSS, JavaScript, Python, Flask, and hosted with render.com

Hosting public domain books from Project Gutenberg with search, filtering, and reading features.

Project Overview

Building on a web scraping assignment from my 'Practical Programming in Python' course, I developed a full-stack web application. It lets users explore and read public domain books from Project Gutenberg, demonstrating responsible web scraping (using official catalog data), database integration, API development, and a dynamic frontend. Users can search, filter by language, browse, and view books directly in the browser.

Skills Learned:

  • Python & Flask: Built the backend with Flask, creating APIs to serve book data and handle search/filter requests.
  • Web Scraping: Scraped Project Gutenberg's catalog responsibly to gather book metadata.
  • Database Management: used SQLlite to store all book metadata
  • Frontend Development: Created a responsive UI with HTML, CSS, and JavaScript for searching, filtering, and reading books.
  • Deployment: Hosted the application on render.com
