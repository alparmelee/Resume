CramCraft

A Modern Flashcard Web Application

This project was developed to provide a more streamlined and functional alternative to traditional study methods, built as a core project for my Understanding Human Computer Interactions course.

Project Overview

A website built for my class "Understanding Human Computer Interactions." I developed the entire front-end from raw HTML, CSS, and JavaScript, collaborating with a small group of people. We utilized GitHub for version control and Supabase to set up the database. Users can log in and create accounts, make their own flashcards for any subject, and share them. Unlike Quizlet, it features a unique option to print flashcards double-sided with terms and definitions flipped, allowing for easy cutting and physical use. It also includes a bulk import option for direct uploads from Quizlet.

Skills Learned:

  • Front-End Development: Built pages and features with plain HTML, CSS, and JavaScript (no frameworks).
  • Version Control: Used GitHub to share and manage code with the team.
  • Database Integration: Connected the site to Supabase for logins and storing user data.
  • User Experience Design: Made the site easy to use, responsive on all devices, and accessible.
  • Problem Solving: Created solutions like double-sided printing and bulk import to handle tricky problems.
